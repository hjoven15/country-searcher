Aplicación sencilla en **React** para buscar países utilizando la **REST Countries API**.  
Permite buscar por nombre y muestra la **bandera**, la **capital** y un **enlace a Google Maps**.

---

## ✨ Características

- 🔎 Búsqueda de países por nombre
- 🏳️ Visualización de la bandera
- 🏙️ Información de la capital
- 📍 Enlace directo a Google Maps
- ⚠️ Alertas informativas en caso de error o país no encontrado

---

## 🚀 Tecnologías

- ⚛️ React
- ⚡ Vite
- 🌐 Axios
- 🌍 REST Countries API
- 🍬 SweetAlert2 (alertas)
- 📏 PropTypes (validación de props)

## ⚙️ Instalación y ejecución

Sigue estos pasos para clonar y ejecutar el proyecto en tu máquina local:

```bash
# 1. Clonar el repositorio
git clone https://github.com/hjoven15/country-searcher.git

# 2. Entrar en el directorio del proyecto
cd country-searcher

# 3. Instalar las dependencias
npm install

# 4. Ejecutar la aplicación en modo desarrollo
npm run dev
```
